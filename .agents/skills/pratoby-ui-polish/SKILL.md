# PratoBy UI Polish

Use esta skill para tarefas visuais e de experiência do usuário.

## Identidade visual

O PratoBy deve parecer:

- SaaS premium;
- moderno;
- confiável;
- claro;
- amigável para pequenos negócios;
- útil para restaurantes, lanchonetes, pizzarias, confeitarias e delivery.

## Paleta e estilo

- Laranja como cor principal.
- Azul-marinho escuro para contraste e autoridade.
- Bege/off-white para fundos quentes.
- Branco limpo em cards.
- Cantos arredondados consistentes.
- Sombras suaves.
- Espaçamento generoso.
- Tipografia limpa e legível.
- Ícones simples e coerentes.
- Aparência profissional, não infantil.

## O que melhorar

Priorize:

- hierarquia visual;
- responsividade mobile;
- contraste;
- CTA;
- espaçamento;
- leitura rápida;
- estados vazios;
- loading;
- erro;
- microcopy;
- consistência entre cards, botões, badges e inputs;
- acessibilidade básica.

## Restrições

Não altere:

- regra de negócio;
- pedidos;
- checkout;
- pagamento;
- plano;
- billing;
- autenticação;
- Firestore;
- Cloud Functions;
- nomes de campos;
- rotas;
- permissões;
- modelo de dados.

Não faça redesign total se o pedido for pequeno.

## Quando usar Taste Skill

Em tarefas visuais, combine com:

```text
Use $pratoby-ui-polish, $gpt-taste e $redesign-existing-projects.
```

Para telas calmas/premium, também pode usar:

```text
Use $high-end-visual-design.
```

Para páginas editoriais como changelog, termos e docs:

```text
Use $minimalist-ui.
```
